**EXITAPP** exits the program. 

```
ExitAPP 
ExitCode (Numeric Expression)
```

### Parameters

**ExitCode** 

Optional. Specifies an exit code. If **ExitCode** is not specified, the default exit code is 0.


### Remarks
**EXITAPP** causes the program to terminate.
