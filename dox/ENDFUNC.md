**ENDFUNC** defines the end of a function. 

```

 **ENDFUNC** 
        
```

### Remarks
**ENDFUNC** must be the last statement within a subroutine that is preceded with a [BEGFUNC](BEGFUNC.html) and also contains a [LEAVESR](LEAVESR.html).
